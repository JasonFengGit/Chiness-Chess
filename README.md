## Features
- accurate move validity detection and warning 
- save and load a certain game/endgame
- piece twinkles when selected
- undo the last move
- background music

## Run
```
git clone https://github.com/JasonFengGit/ChinessChess
cd ChinessChess
ChinessChess.jar
```
